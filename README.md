# Leadsbridge (FINTRIXX)

Welcome to the **Leadsbridge** frontend repository. This project serves as the dedicated landing page for **FINTRIXX**, an Amazon Growth Partner. We specialize in boosting Amazon success with expert SEO, PPC, and comprehensive growth strategies.

## 🚀 Live Website

Check out the live project here: **[https://leadsbridge.vercel.app/](https://leadsbridge.vercel.app/)**

## ✨ Key Features

- **Modern UI/UX:** Clean, professional, and visually appealing interface.
- **Fully Responsive:** Optimized for Mobile, Tablet, and Desktop screens.
- **Service Highlights:** Sections detailing Amazon SEO, PPC campaigns, and overall growth strategies.
- **Conversion Focused:** High-converting custom UI including an interactive mock Calendly strategy session interface.
- **Optimized Performance:** Built with Next.js and styled with Tailwind CSS for lightning-fast speeds.

## 🛠️ Tech Stack

- **Framework:** [Next.js](https://nextjs.org)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Components:** Built with React Functional Components.

## 💻 Getting Started

To run this project locally, simply follow these steps:

1. **Clone the repository:**

   ```bash
   git clone <repository-url>
   cd leadsbridge
   ```

2. **Install dependencies:**

   ```bash
   npm install
   ```

3. **Run the development server:**

   ```bash
   npm run dev
   ```

4. **View the app:**
   Open [http://localhost:3000](http://localhost:3000) in your browser.
